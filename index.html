The decimal system

The decimal system is the most familiar numeric system, as it’s used in everyday human activities. It is essential for general arithmetic, financial transactions, and measurement systems. Its simplicity and compatibility with the way humans count make it the default choice in most non-technical contexts. From calculating the cost of groceries to measuring distance, the decimal system is a cornerstone of human numeracy.

The binary system

The binary system, consisting only of the digits 0 and 1, is the fundamental language of computers. It is used to represent all forms of digital data and instructions in modern computing systems. Binary is the backbone of machine-level programming, enabling devices like CPUs, memory, and storage systems to perform calculations, process logic, and store information. Each binary digit, or "bit," represents a simple on/off state, forming the basis of all complex digital operations.

The octal system

The octal system, which uses digits 0–7, was historically significant in early computing as a shorthand for binary, especially for systems that grouped binary digits into chunks of three. While its use has diminished in modern applications, it is still relevant in Unix-based operating systems. For instance, octal numbers are used to set file permissions via the chmod command. A permission code like 777 means the file is fully accessible: 7 for read, write, and execute permissions for the owner, group, and others, respectively.

The hexadecimal system

The hexadecimal system uses the digits 0–9 and the letters A–F, making it ideal for representing large numbers compactly. It is widely used in computing for tasks such as defining memory addresses, specifying error codes, and encoding binary data in a more human-readable format. A notable application is the representation of RGB colors in digital systems, where two hexadecimal digits define the intensity of red, the next two define green, and the last two define blue. For example, #FF5733 specifies a vivid orange color with maximum red (FF), moderate green (57), and a low level of blue (33).
